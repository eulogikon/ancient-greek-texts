# Mission

**Repository:** `eulogikon/ancient-greek-texts`
**Status:** Authoritative.

---

This repository exists as the data store for the Eulogikon project.

It stores published corpus artifacts as files (PDF now, with Markdown and plain
text to follow) as the Eulogikon project data store.

Generation and rendering are upstream concerns. This repository is the stored
artifact layer only.

---

## Where things live

This repository is the artifact archive only.
The pipeline that produces these files lives upstream in the Eulogikon project.
