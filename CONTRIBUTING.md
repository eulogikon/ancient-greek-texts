# Contributing

This repository is a data store. The files here are generated from the
upstream Eulogikon project and written to this repository as output. They
are not edited by hand.

## Where changes go

- **Text or content corrections** — fix the source upstream in the
  Eulogikon project. The files regenerate from it.
- **Rendering or layout changes** — handled upstream.
- **This repo** only receives regenerated files and updates to these docs.
