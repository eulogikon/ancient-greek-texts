# Fragments on Athenian Dining

*Ἀποσπάσματα περὶ Ἀθηναίων Δείπνων*

**Author:** Lynceus of Samos the Comic Poet

**eul_wid:** `iyi-ab` · [HTML page](https://eulogikon.org/works/lynceus-samos-athenian-dining-iyi-ab)

## Text

### book 1.1

[spk_β]Μάγειρ’ ὁ θύων ἔσθ’ ὁ δειπνίζων τ’ ἐμέ Ῥόδιος, ἐγὼ δ’ ὁ κεκλημένος

### book 1.2

Περίνθιος· οὐδέτερος ἡμῶν ἥδεται

### book 1.3

τοῖς ἀττικοῖς δείπνοις. Β. ἀηδία γάρ ἐστιν ἀττική;

### book 1.1

[ln_5][spk_α]Α. ὥσπερ ξενική· παρέθηκε πίνακα γὰρ μέγαν, ἔχοντα μικροὺς πέντε

### book 1.2

πινακίσκους ἐν οἷ. τούτων ὁ μὲν ἔχει

### book 1.3

σκόροδον, ὁ δ’ ἐχίνους δύο, ὁ δὲ θρυμματίδα γλυκεῖαν, ὁ δὲ κόγχας δέκα, ὁ δ’ ἀντακαίου μικρόν. ἐν ὅσῳ δ’ ἐσθίω,[ln_10]ἕτερος ἐκεῖν’, ἐν ὅσῳ δ’ ἐκεῖνος, τοῦτ’ ἐγώ ἠφάνισα· βούλομαι δ’ ἐγώ, βέλτιστε σύ, κἀκεῖνο καὶ τοῦτ’, ἀλλ’ ἀδύνατα βούλομαι· οὔτε

### book 1.4

στόματα γὰρ οὔτε

### book 1.5

χείλη πέντ’ ἔχω. ὄψιν μὲν οὖν ἔχει τὰ τοιαῦτα

### book 1.6

ποικίλην,

### book 1.1

[ln_15][spk_β]ἀλλ’ οὐδέν ἐστι

### book 1.2

τοῦτο

### book 1.3

πρὸς τὴν γαστέρα· κατέπασα γὰρ τὸ χεῖλος, οὐκ ἐνέπλησα δέ. τί οὖν ἔχεις; Β. ὄστρεια

### book 1.4

πολλά. Α. πίνακα μοι

### book 1.5

τούτων

### book 1.6

παραθήσεις αὐτὸν ἐφ’ ἑαυτοῦ μέγαν. ἔχεις ἐχίνους; Β. ἕτερος ἔσται

### book 1.7

σοι πίναξ·

### book 1.1

[ln_20][spk_α]αὐτὸς γὰρ αὐτὸν ἐπριάμην ὀκτὼ ’βολῶν. Α. ὀψάριον αὐτὸ τοῦτο

### book 1.2

παραθήσεις μόνον, ἵνα

### book 1.3

ταὐτὰ πάντες, μὴ τὸ μὲν ἐγώ, τὸ δ’ ἕτερος.
