# Poem of Naupactus, Fragments

*Ναυπάκτια Ἀποσπάσματα*

**Author:** Epic Work Anonymous

**eul_wid:** `cyy-aa` · [HTML page](https://eulogikon.org/grc/epic-work-anonymous/naupactus-fragments)

## Text

### book 1.1

τὴν δὲ μεθ’ ὁπλοτάτην Ἐριώπην ἐξονόμαζεν, Ἀλκιμάχην δὲ πατήρ

### book 1.2

τε καὶ Ἄδμητος καλέεσκεν.

### book 2.1

ἀλλ’ ὃ μὲν οὖν ἐπὶ θινὶ θαλάσσης εὐρυπόροιο οἰκία ναιετάασκε

### book 2.2

πολύρρην

### book 2.3

πουλυβοώτης.

### book 6.1

δὴ τότ’ ἄρ’ Αἰήτηι πόθον ἔμβαλε δῖ’ Ἀφροδίτη Εὐρυλύτης

### book 6.2

φιλότητι μιγήμεναι, ἧς ἀλόχοιο, κηδομένη

### book 6.3

φρεσὶν ἧισιν, ὅπως μετ’ ἄεθλον Ἰήσων νοστήσηι οἶκόνδε σὺν ἀγχεμάχοις ἑτάροισιν.

### book 7

φευγέμεν ἐκ μεγάροιο θοὴν διὰ νύκτα μέλαιναν[ln_1]* ποδοψοφίην ἀίουσα
