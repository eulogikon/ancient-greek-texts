# Epigram

*Ἐπίγραμμα*

**Author:** Dosiadas of Crete

**eul_wid:** `icu-aa` · [HTML page](https://eulogikon.org/works/dosiadas-crete-epigram-icu-aa)

## Text

### book 26.1.1

Εἱμάρσενός με στήτας

### book 26.1.2

πόσις, μέροψ δίσαβος,

### book 26.1.3

τεῦξ’, οὐ σποδεύνας ἶνις Ἐμπούσας, μόρος

### book 26.1.4

Τεύκροιο βούτα καὶ κυνὸς τεκνώματος,

### book 26.5.1

Χρύσας δ’ ἀίτας, ἆμος ἑψάνδρα

### book 26.5.2

τὸν γυιόχαλκον οὖρον ἔρραισεν,

### book 26.5.3

ὃν ἁπάτωρ δίσευνος

### book 26.5.4

μόγησε ματρόριπτος·

### book 26.5.5

ἐμὸν δὲ τεῦγμ’ ἀθρήσας

### book 26.10.1

Θεοκρίτοιο κτάντας

### book 26.10.2

τριεσπέροιο καύστας

### book 26.10.3

θώυξεν αἴν’ ἰύξας·

### book 26.10.4

χάλεψε γάρ νιν ἰῷ

### book 26.10.5

σύργαστρος ἐκδυγήρας·

### book 26.15.1

τὸν δ’ αἰλινεῦντ’ ἐν ἀμφικλύστῳ

### book 26.15.2

Πανός τε ματρὸς εὐνέτας, φὼρ

### book 26.15.3

δίζῳος, ἶνίς τ’ ἀνδροβρῶτος Ἰλιορραιστᾶν

### book 26

ἦρ’ ἀρδίων ἐς Τευκρίδ’ ἄγαγον τριπόρθητον.
