# Sympotic Fragments

*Ἀποσπάσματα*

**Author:** Nicochares the Comic Poet

**eul_wid:** `fcs-ab` · [HTML page](https://eulogikon.org/works/nicochares-sympotic-fragments-fcs-ab)

## Text

### book 1.1

Οἰνόμαος οὗτος

### book 1.2

χαῖρε, πέντε καὶ δύο, κἀγώ τε καὶ σὺ συμπόται γενοίμεθα.

### book 1.1

Σοφαῖσι

### book 1.2

παλάμαις

### book 1.3

τεκτόνων εἰργασμένον καὶ πόλλ’ ἐν αὑτῷ λέπτ’ ἔχον καδίσκια κυμινοδόκον.

### book 2.1

Τί δῆτ’; ἀπαιδευτότερος εἶ Φιλωνίδου

### book 2.2

τοῦ Μελιτέως.

### book 1.1

Φέρε νῦν

### book 1.2

ταχέως

### book 1.3

χιτῶνα τόνδ’ ἐπενδύτην.

### book 1.1

Τριχίας δὲ καὶ τὰς

### book 1.2

πρημνάδας τὰς θυννίδας ἐπὶ δεῖπνον ἡκούσας ὑπερπληθεῖς.

### book 1

Εἰς αὔριον δ’ ἀντὶ ῥαφάνων ἑψήσομεν βαλάνιον, ἵνα νῷν ἐξάγῃ τὴν κραιπάλην.

### book 3.1

Εἰ πεύσομαι τὸν ἀηδόνειον ὕπνον ἀποδαρθόντα

### book 3.2

σε, αὐτὸς

### book 3.3

σεαυτὸν αἰτιῶ.
