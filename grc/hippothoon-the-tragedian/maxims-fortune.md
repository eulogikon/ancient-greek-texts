# Maxims on Human Fortune

*Ἀποσπάσματα*

**Author:** Hippothoon the Tragedian

**eul_wid:** `cxu-aa` · [HTML page](https://eulogikon.org/grc/hippothoon-the-tragedian/maxims-fortune)

## Text

### book 1

ἄνθρωπος ὢν μέμνησο τῆς κοινῆς τύχης

### book 2.1

φθόνος κάκιστος κἀδικώτατος θεός, κακοῖς

### book 2.2

τε χαίρει κἀγαθοῖς ἀλγύνεται

### book 3.1

ζευχθεὶς γάμοισιν οὐκέτ’ ἔστ’ ἐλεύθερος. ἀλλ’ ἕν γ’ ἔχει

### book 3.2

τι χρηστόν· ἐν κήδει γὰρ ὤν ἐσθλῷ δέδοικε μηδὲν ἐξαμαρτάνειν

### book 3a?.1

ἄριστον ἀνδρὶ κτῆμα

### book 3a?.2

συμπαθὴς γυνή

### book 4.1

παίδων κρατεῖν δεῖ τῶν νεωτέρων

### book 4.2

σοφούς

### book 5

γῆρας λεόντων κρεῖσσον ἀκμαίων νεβρῶν

### book 6*.1

γάμος κράτιστός ἐστιν ἀνδρὶ σώφρονι, τρόπον γυναικὸς

### book 6*.2

χρηστὸν ἔνδον λαμβάνειν· αὕτη γὰρ ἡ προὶξ οἰκίαν σῴζει μόνη
