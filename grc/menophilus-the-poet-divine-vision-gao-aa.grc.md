# Fragments on Divine Vision

**Author:** Menophilus the Poet

## Text

### book 558.1

[ln_1]Εὐρώπην Λιβύην

### book 558.2

τε καὶ Ἀσίδα πᾶσαν ἀμείψας θαύμασα μυρία καλὰ πολυπλαν〈ί〉ης ὑπὸ λυγρῆς, ἀλλ’ οὔπω

### book 558.3

τοιοῦτον ἴδον σέλας, οὐδ’ ἐν Ὀλύμπῳ αὐτοὺς ἀθανάτους 〈περ〉 ὀΐομαι ἶσον ἰδέσθαι,[ln_5]οἷον ἄελπτον ἄπιστον ἐμὸν νόον ἥρπασε φάσμα καρτερόν, οὔ τι

### book 558.4

φατειόν· ὑπ’ ἀμφασίῃ δ’ ἀλεγεινῇ θυμὸς ἄδην

### book 558.5

πεπότητο, λύθεν δέ μοι ἅψεα πάντα ἐκ κεφαλῆς εἰς ἴχνος, ἀπώλετο δ’ Ἑλλὰς ἅπασα ἐκ

### book 558.6

στηθέων, καὶ πάντα

### book 558.7

χαμαὶ πέσεν, ὅσσα

### book 558.8

περ ἔτλην[ln_10]ὑγρῇ τε

### book 558.9

τραφερῇ τε κυλινδόμενος

### book 558.10

περὶ νόστον. τόσσον γὰρ

### book 558.11

περὶ θυμὸν ἀπείριτον ἵκετο θάμβος. μέλψαι δὲ μνήσειας ἀειθαλέας

### book 558.12

πλοκαμῖδας, οἵαις κυδιόωσαν ἀπ’ ὀλβίστων

### book 558.13

σε λοετρῶν

### book 558.14

φαιδρὴν εἶδον ἅπασαν ἐειδομένην

### book 558.15

Χαρίτεσσιν[ln_15]ἐρχομέναις

### book 558.16

πρὸς Ὄλυμπον Ἀκιδαλίης ἀπὸ πηγῆς.
