# Fragments on Parasites and Feasting

*Ἀποσπάσματα περὶ Παρασίτων καὶ Εὐωχίας*

**Author:** Crobylus the Comic Poet

**eul_wid:** `lhc-aa` · [HTML page](https://eulogikon.org/grc/crobylus-the-comic-poet/parasites-feasting)

## Text

### book 1.1

παράσιτον αὐτόσιτον· αὑτὸν γοῦν

### book 1.2

τρέφων τὰ πλεῖστα

### book 1.3

συνερανιστὸς εἶ τῷ δεσπότῃ.

### book 2.1

καὶ σκάφην λαβών

### book 2.2

τινα τῶν ἐσχαριτῶν τῶν καθαρῶν

### book 3.1

τὸ γὰρ ἐνδελεχῶς μεθύειν τίν’ ἡδονὴν ἔχει, ἀποστεροῦντα ζῶνθ’ ἑαυτὸν

### book 3.2

τοῦ φρονεῖν, ὃ μέγιστον ἡμῶν ἀγαθὸν ἔσχεν ἡ φύσις;

### book 4.1

πάλιν ἡ τοῦ βίου ὑγρότης με

### book 4.2

τοῦ σοῦ τέθαιφε· τὴν ἀσωτίαν ὑγρότητα γὰρ νῦν

### book 4.3

προσαγορεύουσίν

### book 4.4

τινες.

### book 5.1

[spk_β]Λάχης. Β. ἐγὼ δὲ πρὸς σέ. Α. πρόαγε. Β. ποῖ; Α. ὅποι μ’ ἐρωτᾷς; ὡς

### book 5.2

Φιλουμένην, παρ’ ᾗ τἀπιδόσιμ’ ἡμῖν ἐστιν, ἧς ἐχθὲς

### book 5.3

πιεῖν κυάθους ἕκαστον ἐβιάσω σὺ δώδεκα.

### book 6.1

εἰσῆλθεν ἡμίκραιρα

### book 6.2

τακερὰ δέλφακος· ταύτης μὰ τὸν Δί’ οὐχὶ κατέλιπον δ’ ἐγὼ οὐδέν.

### book 7.1

καὶ πλεκτάνην

### book 7.2

στιφρὰν

### book 7.3

σφόδρ’, ἐν

### book 7.4

τούτοις τέ που αἰσχυνόμενον ἧπαρ καπρίσκου

### book 7.5

σκατοφάγου.

### book 8.1

[spk_β]ἐγὼ δὲ πρὸς τὰ θερμὰ ταῦθ’ ὑπερβολῇ τοὺς δακτύλους δήπουθεν Ἰδαίους ἔχω, καὶ τὸν λάρυγγ’ ἥδιστα

### book 8.2

πυριῶ τεμαχίοις. Β. κάμινος, οὐκ ἄνθρωπος.

### book 9.1

χλωρὸν ἐρέβινθόν

### book 9.2

τινα ἐκοττάβιζον κενὸν ὅλως. Β. τράγημα δὲ ἔστιν

### book 9.3

πιθήκου

### book 9.4

τοῦτο δήπου δυστυχοῦς.

### book 10.1

τὸν

### book 10.2

πλακοῦντα κόμμιδι, οὐ μέλιτι διεκόσμει.
