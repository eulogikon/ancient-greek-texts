# Fragments on Parasites and Excess

*Ἀποσπάσματα περὶ Παρασίτων καὶ Ὑπερβολῆς*

**Author:** Crobylus the Comic Poet

**eul_wid:** `lhc-ab` · [HTML page](https://eulogikon.org/grc/crobylus-the-comic-poet/parasites-excess)

## Text

### book 1.1

Παράσιτον αὐτόσιτον· αὑτὸν γοῦν

### book 1.2

τρέφων τὰ πλεῖστα

### book 1.3

συνερανιστὸς εἶ τῷ δεσπότῃ.

### book 2.1

Καὶ σκάφην λαβών

### book 2.2

τινα τῶν ἐσχαριτῶν τῶν καθαρῶν.

### book 1.1

Πάλιν ἡ τοῦ βίου ὑγρότης με

### book 1.2

τοῦ σοῦ τέθαφε· τὴν ἀσωτίαν ὑγρότητα γὰρ νῦν

### book 1.3

προσαγορεύουσιν

### book 1.4

τινές.

### book 2.1

Τὸ γὰρ ἐνδελεχῶς μεθύειν τίν’ ἡδονὴν ἔχει, ἀποστεροῦντα ζῶνθ’ ἑαυτὸν

### book 2.2

τοῦ φρονεῖν, ὃ μέγιστον ἡμῶν ἀγαθὸν ἔσχεν ἡ φύσις;

### book 1.1

[spk_β]Λάχης. Β. ἐγὼ δὲ πρὸς σέ. Α. πρόαγε. Β. ποῖ; Α. ὅποι μ’ ἐρωτᾶς; ὡς

### book 1.2

Φιλουμένην, παρ’ ᾗ τἀπιδόσιμ’ ἡμῖν ἐστιν, ἧς ἐχθὲς

### book 1.3

πιεῖν κυάθους ἕκαστον ἐβιάσω σὺ δώδεκα.

### book 2.1

Καὶ πλεκτάνην

### book 2.2

στιφρὰν

### book 2.3

σφόδρ’, ἐν

### book 2.4

τούτοις τέ που αἰσχυνόμενον ἧπαρ καπρίσκου

### book 2.5

σκατοφάγου.

### book 3.1

Εἰσῆλθεν ἡμίκραιρα

### book 3.2

τακερὰ δέλφακος· ταύτης μὰ τὸν Δί’ οὐχὶ κατέλιπον δ’ ἐγώ οὐδέν.

### book 1.1

Ἐγὼ δὲ πρὸς τὰ θερμὰ ταῦθ’ ὑπερβολῇ τοὺς δακτύλους δήπουθεν Ἰδαίους ἔχω, καὶ τὸν λάρυγγ’ ἥδιστα

### book 1.2

πυριῶ τεμαχίοις.

### book 1

[spk_β]Β. κάμινος, οὐκ ἄνθρωπος.

### book 2.1

Χλωρὸν ἐρέβινθόν

### book 2.2

τινα ἐκοττάβιζον κενὸν ὅλως. Β. τράγημα δέ ἔστιν

### book 2.3

πιθήκου

### book 2.4

τοῦτο δήπου δυστυχοῦς.

### book 3.1

Τὸν

### book 3.2

πλακοῦντα κόμμιδι, οὐ μέλιτι, διεκόσμει.
