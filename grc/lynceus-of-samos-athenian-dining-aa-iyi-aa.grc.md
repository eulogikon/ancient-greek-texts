# Fragments on Athenian Dining

*Ἀποσπάσματα περὶ Ἀθηναίων Δείπνων*

**Author:** Lynceus of Samos the Comic Poet

**eul_wid:** `iyi-aa` · [HTML page](https://eulogikon.org/grc/lynceus-of-samos/athenian-dining-aa)

## Text

### book 1.1

μάγειρ’, ὁ θύων ἔσθ’ ὁ δειπνίζων τ’ ἐμὲ Ῥόδιος, ἐγὼ δ’ ὁ κεκλημένος

### book 1.2

Περίνθιος· οὐδέτερος ἡμῶν ἥδεται

### book 1.3

τοῖς Ἀττικοῖς δείπνοις. ἀηδία γάρ ἐστιν Ἀττικὴ[ln_5]ὥσπερ ξενική· παρέθηκε πίνακα γὰρ μέγαν, ἔχοντα μικροὺς πέντε

### book 1.4

πινακίσκους ἐν οἷ. τούτων ὁ μὲν ἔχει

### book 1.5

σκόροδον, ὁ δ’ ἐχίνους δύο, ὁ δὲ θρυμματίδα γλυκεῖαν, ὁ δὲ κόγχας δέκα, ὁ δ’ ἀντακαίου μικρόν. ἐν ὅσῳ δ’ ἐσθίω,[ln_10]ἕτερος ἐκεῖν’, ἐν ὅσῳ δ’ ἐκεῖνος, τοῦτ’ ἐγὼ ἠφάνισα· βούλομαι δ’ ἐγώ, βέλτιστε σύ, κἀκεῖνο καὶ τοῦτ’, ἀλλ’ ἀδύνατα βούλομαι·

### book 1.1

οὔτε

### book 1.2

στόματα γὰρ οὔτε

### book 1.3

χείλη πέντ’ ἔχω. ὄψιν μὲν οὖν ἔχει τὰ τοιαῦτα

### book 1.4

ποικίλην,

### book 1.1

[ln_15][spk_β]ἀλλ’ οὐδέν ἐστι

### book 1.2

τοῦτο

### book 1.3

πρὸς τὴν γαστέρα· κατέπασα γὰρ τὸ χεῖλος, οὐκ ἐνέπλησα δέ. Β. τί οὖν; Α. ἔχεις ὄστρεια; Β. πολλά. Α. πίνακά μοι

### book 1.4

τούτων

### book 1.5

παραθήσεις αὐτὸν ἐφ’ ἑαυτοῦ μέγαν. ἔχεις ἐχίνους; Β. ἕτερος ἔσται

### book 1.6

σοι πίναξ·

### book 1.1

[ln_20][spk_α]αὐτὸς γὰρ αὐτὸν ἐπριάμην ὀκτὼ ’βολῶν. Α. ὀψάριον αὐτὸ τοῦτο

### book 1.2

παραθήσεις μόνον, ἵνα

### book 1.3

ταὐτὰ πάντες, μὴ τὸ μὲν ἐγώ, τὸ δ’ ἕτερος.
