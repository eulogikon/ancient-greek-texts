# Epigrams

*Ἐπιγράμματα*

**Author:** Phaedimus of Bisanthe

**eul_wid:** `jro-aa` · [HTML page](https://eulogikon.org/grc/phaedimus-of-bisanthe/epigrams)

## Text

### book 271.1.1

Ἄρτεμι, σοὶ τὰ πέδιλα Κιχησίου εἵσατο υἱὸς

### book 271.1.2

καὶ πέπλων ὀλίγον πτύγμα Θεμιστοδίκη,

### book 271.1.3

οὕνεκά οἱ πρηεῖα λεχοῖ δισσὰς ὑπερέσχες

### book 271.1.4

χεῖρας ἄτερ τόξου, πότνια, νισσομένη.

### book 271.5.1

Ἄρτεμι, νηπίαχον δὲ καὶ εἰσέτι παῖδα Λέοντος

### book 271.5.2

νεῦσον ἰδεῖν κοῦρον γυῖ’ ἐπαεξόμενον.

### book 739.1.1

Αἰάζω Πολύανθον, ὃν εὐνέτις, ὦ παραμείβων,

### book 739.1.2

νυμφίον ἐν τύμβῳ θῆκεν Ἀρισταγόρη

### book 739.1.3

δεξαμένη σποδιήν τε καὶ ὀστέα—τὸν δὲ δυσαὲς

### book 739.1.4

ὤλεσεν Αἰγαίου κῦμα περὶ Σκίαθον—,

### book 739.5.1

δύσμορον ὀρθρινοί μιν ἐπεὶ νέκυν ἰχθυβολῆες,

### book 739.5.2

ξεῖνε, Τορωναίων εἵλκυσαν ἐς λιμένα.

### book 2.1.1

Καλλίστρατός σοι, Ζηνὸς ὦ διάκτορε,

### book 2.1.2

ἔθηκε μορφῆς ξυνὸν ἥλικος τύπον·

### book 2.1.3

Κηφισιεὺς ὁ κοῦρος· ᾧ χαρείς, ἄναξ,

### book 2.1.4

Ἀπολλοδώρου παῖδα καὶ πάτραν σάω.

### book 22.1.1

Τόξον μέν, ᾧ Γίγαντος ὤλεσας σθένος,

### book 22.1.2

ἴσχε βίης, ἑκάεργ’ ἀνάσσων·

### book 22.1.3

οὔ σοι φαρέτρη λύεται λυκοκτόνος·

### book 22.1.4

τοῖσδε δ’ ἐπ’ ἠιθέοις ὀιστὸν

### book 22.5.1

στρέφειν Ἔρωτος, τόφρ’ ἀλέξωνται πάτρῃ

### book 22.5.2

θαρσαλέοι φιλότατι κούρων·

### book 22.5.3

πυρὸς γὰρ ἀλκή, καὶ θεῶν ὑπέρτατος

### book 22.5.4

αἰὲν ὅδε προμάχους ἀέξειν.

### book 22.5.5

Μελιστίωνος δ’, ὦ πατρώιον σέβας

### book 22.10

Σχοινιέων, ἐπίηρα δέχθαι.
