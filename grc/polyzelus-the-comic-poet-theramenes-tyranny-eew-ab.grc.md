# Fragments on Theramenes and Tyranny

*Ἀποσπάσματα περὶ Θηραμένους καὶ Τυραννίδος*

**Author:** Polyzelus the Comic Poet

**eul_wid:** `eew-ab` · [HTML page](https://eulogikon.org/grc/polyzelus-the-comic-poet/theramenes-tyranny)

## Text

### book 1.1

Τριῶν κακῶν γοῦν ἦν ἑλέσθ’ αὐτῷ τι πᾶς’ ἀνάγκη, ἢ ξύλον ἐφέλκειν, ἢ πιεῖν κώνειον, ἢ προδόντα τὴν ναῦν ὅπως τάχιστα τῶν κακῶν ἀπαλλαγῆναι. ταῦτ’ ἔστι

### book 1.2

τρία Θηραμένους, ἅ σοι

### book 1.3

φυλακτέ’ ἐστί.

### book 1

Οὗπερ αἱ χύτραι κρέμανται καὶ τὸ φρύγετρον.

### book 1.1

Ἱερὸν γὰρ ὂν

### book 1.2

τετύχηκας Ἐπακρίου Διός.

### book 2.1

Ὑψιπέταλοί τε κράμβαι

### book 2.2

συχναί.

### book 3.1

Ὥσπερ

### book 3.2

Χαλκιδικὴ τέτοκεν ἡμῖν ἡ γυνή.

### book 1.1

Ὁ μαινόμενος ἐκεινοσὶ Διονύσιος

### book 1.2

χρυσοῦν ἔχων

### book 1.3

χλίδωνα καὶ τρυφήματα ἐν τῷ μύρῳ παρ’ Ἀθηναίων μακαρίζεται.

### book 2

Ἀλλ’ οὐ πρᾳότερα τὰ φθέγματα οὐδὲ γλυκίονα.
