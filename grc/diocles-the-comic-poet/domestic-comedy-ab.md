# Domestic Comedy Fragments

**eul_wid:** `dam-ab` · [HTML page](https://eulogikon.org/grc/diocles-the-comic-poet/domestic-comedy-ab)

## Text

### book 1.1

Ὑδρία

### book 1.2

τις ἢ χαλκοῦς

### book 1.3

ποδανιπτὴρ ἢ λέβης.

### book 2

Πλυνεῖ τε τὰ κακὰ τῶν κακῶν ὑμᾶς.

### book 1

Ἅλλεται δ’ ὑφ’ ἡδονῆς κεστρεύς.

### book 1

Καὶ διὰ τετρημένων ἀθέλδεται τύπων.

### book 2

Πῶς δὲ καὶ κεκραμένον

### book 2

[spk_β]πίνειν τὸν οἶνον δεῖ με; Β. τέτταρα καὶ δύο.

### book 3

Ἀπὸ λασάνων θερμὴν ἀφαιρήσω χύτραν.

### book 4-5

Ἡ μίλτος οἶμαι καὶ τὸ τιγγάβαρι. Τιγγάβαρι καὶ μίλτος ἀναμεμιγμένη.

### book 1.1

Μηδείς

### book 1.2

ποθ’ ὑμῶν, ἄνδρες, ἐπιθυμησάτω γέρων γενέσθαι, προνοησάτω δ’ ὅπως νέος ὢν ἀγαθόν

### book 1.3

τι τῇ ψυχῇ παθών ὥρᾳ καταλύσει, μηδ’ ἀγόμφιόν

### book 1.4

ποτε[ln_5]αἰῶνα

### book 1.5

τρίψει.
