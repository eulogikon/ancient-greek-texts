# Fragments of Comic Courtesans

*Ἀποσπάσματα Ἑταιρῶν Κωμικῶν*

**Author:** Phoenicides the Comic Poet

**eul_wid:** `jsi-aa` · [HTML page](https://eulogikon.org/works/phoenicides-comic-courtesans-jsi-aa)

## Text

### book 1.1

[spk_β]δύνασαι

### book 1.2

σιωπᾶν; Β. ὥστε

### book 1.3

τοὺς τὰς διαλύσεις

### book 1.4

συντιθεμένους κεκραγέναι δοκεῖν.

### book 2.1

[spk_β]μύρτων λέγουσιν καὶ μέλιτος ἐγκώμια, καὶ τῶν

### book 2.2

προπυλαίων καὶ τέταρτον ἰσχάδων· τούτων ἐγευσάμην καταπλεύσας εὐθέως— Β. καὶ τῶν

### book 2.3

προπυλαίων; Α. κοὐδὲν ἦν

### book 2.4

τούτων ὅλως[ln_5]πρὸς ἀτταγῆνα

### book 2.5

συμβαλεῖν τῶν βρωμάτων.

### book 3.1

τρίτον δὲ πρὸς

### book 3.2

τούτοισι τὸν

### book 3.3

σοφώτατον

### book 3.4

Χαίριππον. οὗτος, ὥσπερ οἶδας, ἐσθίει μέχρι ἂν διδῷ τις ἢ λάθῃ διαρραγείς. τοιοῦτ’ ἔχει

### book 3.5

ταμιεῖον ὥσπερ οἰκίας.

### book 4.1

μὰ τὴν Ἀφροδίτην οὐκ ἂν ὑπομείναιμ’ ἔτι, Πυθιάς, ἑταιρεῖν. χαιρέτω· μή μοι λέγε· ἀπέτυχον· οὐδὲν

### book 4.2

πρὸς ἐμέ· καταλῦσαι θέλω. εὐθὺς ἐπιχειρήσασα φίλον ἔσχον

### book 4.3

τινὰ[ln_5]στρατιωτικόν· διαπαντὸς οὗτος τὰς μάχας ἔλεγεν, ἐδείκνυ’ ἅμα λέγων τὰ τραύματα, εἰσέφερε δ’ οὐδέν. δωρεὰν ἔφη

### book 4.4

τινὰ παρὰ τοῦ βασιλέως λαμβάνειν, καὶ ταῦτ’ ἀεὶ ἔλεγεν· διὰ ταύτην ἣν λέγω τὴν δωρεὰν[ln_10]ἐνιαυτὸν ἔσχε μ’ ὁ κακοδαίμων δωρεάν. ἀφῆκα

### book 4.5

τοῦτον, λαμβάνω δ’ ἄλλον

### book 4.6

τινά, ἰατρόν. οὗτος εἰσάγων

### book 4.7

πολλούς

### book 4.8

τινας ἔτεμν’, ἔκαε, πτωχὸς ἦν καὶ δήμιος. δεινότερος οὗτος θατέρου μοι κατεφάνη.

### book 4.1

[ln_15]ὁ μὲν διήγημ’ ἔλεγεν, ὁ δ’ ἐποίει νεκρούς. τρίτῳ συνέζευξ’ ἡ τύχη με

### book 4.2

φιλοσόφῳ, πώγων’ ἔχοντι καὶ τρίβωνα καὶ λόγον. εἰς

### book 4.3

προὖπτον ἦλθον ἐμπεσοῦσα δὴ κακόν· οὐδὲν ἐδίδου γάρ· εἴ τι δ’ αἰτοίην, ἔφη[ln_20]οὐκ ἀγαθὸν εἶναι τἀργύριον. ἔστω κακόν, διὰ τοῦτο δός μοι, ῥῖψον· οὐκ ἐπείθετο.
