# Cook's Boastful Fragments

*Ἀπόσπασμα*

**Author:** Demetrius the Younger 

**eul_wid:** `khs-aa` · [HTML page](https://eulogikon.org/grc/demetrius-the-younger/cooks-boastful-fragments)

## Text

### book 1.1

σύ μου καταφρονεῖς, ὅτι μάγειρός εἰμ’, ἴσως· ὅσον δ’ ἀπὸ ταύτης τῆς τέχνης εἴργασμ’ ἐγώ, οὐδεὶς ὑποκριτής ἐσθ’ ὅλως εἰργασμένος. καπνιζομένη

### book 1.2

τυραννὶς αὕτη ’σθ’ ἡ τέχνη.[ln_5]ἀβυρτακοποιὸς

### book 1.3

παρὰ Σέλευκον ἐγενόμην· παρ’ Ἀγαθοκλεῖ δὲ πρῶτος εἰσήνεγκ’ ἐγὼ τῷ Σικελιώτῃ τὴν

### book 1.4

τυραννικὴν

### book 1.5

φακῆν. τὸ μέγιστον οὐκ εἴρηκα· Λαχάρους

### book 1.6

τινός, ὅτ’ ἦν ὁ λιμός, ἑστιῶντος

### book 1.7

τοὺς φίλους,

### book 1

[ln_10][spk_β]ἀνάληψιν ἐποίης’ εἰσενέγκας κάππαριν. Β. γυμνὴν Ἀθηνᾶν τότ’ ἐποίησε Λαχάρης οὐδὲν ἐνοχλοῦσαν· σὲ δ’ ἐνοχλοῦντα νῦν ἐγώ.
