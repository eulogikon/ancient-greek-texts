# Comic Fragments on Human Folly

*Ἀποσπάσματα περὶ Ἀνθρωπίνης Μωρίας*

**Author:** Sosicrates the Comic Poet 

**eul_wid:** `kyq-ab` · [HTML page](https://eulogikon.org/works/sosicrates-comic-fragments-kyq-ab)

## Text

### book 1.1

Ὅταν γάρ, οἶμαι, λευκὸς ἄνθρωπος, παχύς, ἀργός, λάβῃ δίκελλαν εἰωθὼς

### book 1.2

τρυφᾶν

### book 1.3

πενταστάτηρον, γίγνεται τὸ πνεῦμ’ ἄνω.

### book 1.1

Λεπτὴ δὲ κυρτοῖς ἐγγελῶσα κύμασιν αὔρα, κόρη

### book 1.2

Σκείρωνος, ἡσύχῳ ποδί προσῆγε

### book 1.3

πράως καὶ καλῶς τὸν κάνθαρον.

### book 1.1

Ἀγαθοὶ δὲ τὸ κακόν ἐσμεν ἐφ’ ἑτέρων ἰδεῖν, αὐτοὶ δ’ ὅταν

### book 1.2

ποιῶμεν οὐ γιγνώσκομεν.

### book 3.1

Ὅταν

### book 3.2

ποθεῖν λέγῃ σε καὶ στέργειν γυνή, φοβοῦ παρ’ αὐτῆς

### book 3.3

πλείον’ ὧν λέγει κακά.
