# Symposium and Drinking Fragments

*Ἀποσπάσματα*

**Author:** Clearchus the Comic Poet

## Text

### book 1.1

Τήνδ’ ἐγώ μεστὴν ἅπαξ ἐπονομάσας

### book 1.2

προπίομαι

### book 1.3

συγγενέσι πίστωμα

### book 1.4

φιλίας, πιὼν δ’ ἐρῶ

### book 1

[spk_β]τὰ λοιπά· πνίγομαι γάρ. Β. ἀλλ’ ἐπιρρόφει.

### book 2.1

Γόγγρων

### book 2.2

τε λευκῶν πᾶσι

### book 2.3

τοῖς κολλώδεσι βρόχθιζε

### book 2.4

τούτοις γὰρ

### book 2.5

τρέφεται τὸ πνεῦμα καί τὸ φωνάριον ἡμῶν

### book 2.6

περίσαρκον γίγνεται.

### book 1.1

Εἰ τοῖς μεθυσκομένοις ἑκάστης ἡμέρας ἀλγεῖν

### book 1.2

συνέβαινε τὴν κεφαλὴν

### book 1.3

πρὸ τοῦ πιεῖν τὸν ἄκρατον, ἡμῶν οὐδὲ εἷς ἔπινεν ἄν. νυνὶ δὲ πρότερον

### book 1.4

τοῦ πόνου τὴν ἡδονήν[ln_5]προλαμβάνοντες ὑστεροῦμεν τἀγαθοῦ.

### book 1.1

[spk_β]Λάβ’ ὕδωρ κατὰ χειρός. Β. μηδαμῶς. καλῶς ἔχει. Α. λάβ’ ὦγάθ’. οὐδὲν

### book 1.2

χεῖρον. ἡ παῖς ἐπιτίθει ἐπὶ τὴν

### book 1.3

τράπεζαν κάρυα καὶ τραγήματα.

### book 1

Οἶνος Λέσβιος, ὃν αὐτὸς ἐποίησεν ὁ Μάρων ἐμοὶ δοκῶ.
