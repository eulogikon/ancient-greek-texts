# Fragments on Fish and Feasting

## Text

### book 1

Καὶ ταῦτ’ ἔχων τὸ ῥύγχος οὑτωσὶ μακρόν.

### book 2.1

Τίς ἐκέρασε

### book 2.2

σφῶν, ὦ κακόδαιμον, ἴσον ἴσῳ;

### book 1.1

Ἀνδρῶν ἄριστος καὶ μάλιστ’ ἐμοὶ ξένος, ἀτὰρ

### book 1.2

παρ’ ἐμοί γ’ ὢν εἶχεν οὐδὲ σύμβολον.

### book 2.1

Ταδὶ δ’ ἅμα

### book 2.2

χοίρων ἀκροκώλια μικρῶν, ταύρου τ’ αὐξίκερω

### book 2.3

### book 2.4

φλογίδες.

### book 3.1

Τὸ πῦρ

### book 3.2

πολὺ λίαν· ὕπαγε τὰς ἀκροβελίδας ἄκρας.

### book 1.1

Αἰγύπτιος μιαρώτατος τῶν ἰχθύων κάπηλος, Ἕρμαιος, ὃς βίᾳ δέρων ῥίνας γαλεούς

### book 1.2

τε πωλεῖ καὶ τοὺς λάβρακας ἐντερεύων, ὡς λέγουσιν ἡμῖν.

### book 3.1

Αἱρουμένους

### book 3.2

ποιῶμεν

### book 3.4

### book 3.5

παλιναιρέτους.

### book 9.1

### book 9.2
