# Epigrams

*Ἐπιγράμματα*

**Author:** Mesomedes of Crete

**eul_wid:** `pfu-ab` · [HTML page](https://eulogikon.org/works/mesomedes-crete-epigrams-pfu-ab)

## Text

### book 63.1.1

Ἕρπουσα, βεβῶσα, ποτωμένα,

### book 63.1.2

νόθον ἴχνος ἀειρομένα δρομάς,

### book 63.1.3

πτερόεσσα μὲν ἦν τὰ πρόσω γυνά,

### book 63.1.4

τὰ δὲ μέσσα βρέμουσα λέαινα θήρ,

### book 63.5.1

τὰ δ’ ὄπισθεν ἑλισσόμενος δράκων.

### book 63.5.2

οὔθ’ ὁλκὸς ἀπέτρεχεν, οὐ γυνά,

### book 63.5.3

οὔτ’ ὄρνις ὅλον δέμας οὔτε θήρ·

### book 63.5.4

κούρα γὰρ ἐφαίνετ’ ἄνευ ποδῶν,

### book 63.5.5

κεφαλὰν δ’ οὐκ ἔσχε βρέμουσα θήρ.

### book 63.10

φύσιν εἶχεν ἄτακτα κεκραμέναν·

### book 63

ἀτέλεστα τέλεια μεμιγμένα.

### book 323.1.1

Τὰν ὕελον ἐκόμιζε

### book 323.1.2

κόψας ἐργάτας ἀνήρ·

### book 323.1.3

ἐς δὲ πῦρ ἔθηκε βῶλον,

### book 323.1.4

ὡς σίδηρον εὐσθενῆ·

### book 323.5.1

ἁ δ’ ὕελος οἷα κηρὸς

### book 323.5.2

ἐξεχεῖτο παμφάγοισι

### book 323.5.3

φλοξὶν ἐκπυρουμένα·

### book 323.5.4

θαῦμα δ’ ἦν ἰδεῖν βροτοῖς

### book 323.5.5

ὁλκὸν ἐκ πυρὸς ῥέοντα

### book 323.10.1

καὶ τὸν ἐργάτην τρέμοντα,

### book 323.10.2

μὴ πεσὼν διαρραγῇ·

### book 323.10.3

ἐς δὲ διπτύχων ἀκμὰς

### book 323

χηλέων ἔθηκε βῶλον ...
